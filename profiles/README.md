# Profils KYS

## Objet

Les profils KYS — Know Your Self — définissent des projections certifiées, limitées et finalisées d’un corpus personnel, familial, institutionnel ou patrimonial.

Ils ne constituent pas un accès général à une personne, à un Cogentigram ou à un jumeau numérique. Ils définissent au contraire ce qui peut être exposé, à quelle fin, sous quelles conditions, avec quels droits de contestation, de révocation et d’audit.

## Principe

Un profil KYS doit toujours expliciter :

- la finalité autorisée ;
- les données ou traces accessibles ;
- les usages interdits ;
- les règles de consentement ;
- les règles de révocation ;
- les règles de contestation ;
- les règles de publication ;
- les règles posthumes éventuelles ;
- la structure fiduciaire compétente, si elle existe.

## Séparation des usages

Un même corpus peut donner lieu à plusieurs profils KYS distincts.

Exemples :

- `KYS Genealogy` : usage généalogique ;
- `KYS Legacy` : usage de transmission posthume ;
- `KYS Family` : usage familial restreint ;
- `KYS Public` : usage public documenté ;
- `KYS Research` : usage de recherche encadré ;
- `KYS Mandate` : usage lié à un mandat explicite.

Chaque profil doit rester finalisé. Un accès accordé pour une finalité ne vaut pas autorisation générale.

## Non-substitution

Un profil KYS ne doit pas permettre de confondre une personne avec sa représentation numérique.

Il peut aider à vérifier, documenter, transmettre ou interroger un corpus. Il ne doit pas permettre de faire agir, parler, voter, contracter ou décider une personne sans mandat explicite.

## Lien avec PrivAI

Les profils KYS prolongent la charte PrivAI. Ils traduisent ses principes dans des contrats d’usage concrets : consentement, non-extraction, souveraineté personnelle, traçabilité, révocabilité, contestabilité, non-substitution, gouvernance posthume et protection contre la capture.
